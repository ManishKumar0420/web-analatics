

## Getting Started

First, run the development server:

```bash
npm i
or
npm install

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Open [http://localhost:3000/test](http://localhost:3000/test) with your browser for using and recording the analytics 


 I have used next.js for my back end and frontend and MongoDB for database
